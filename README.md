## My NeoVim 

> Requires
- Neovim
- Node 12+ for copilot

> Setup

```bash
git clone https://github.com/ahmedsaheed/neovim.git ~/.config/nvim
```

```bash
# Setup Copilot: Once cloned
nvim
Copilot Setup
```
